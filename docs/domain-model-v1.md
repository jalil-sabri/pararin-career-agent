# Domain Model V1

## Purpose

Define the main domain concepts used in the first version of the system.

## Core Concepts

### Candidate

A person who evaluates job postings and applies for suitable roles.

### JobPosting

A job posting submitted for analysis.

### CandidateProfile

The candidate's professional profile used for matching.

### ResumeVersion

A resume variant used for a specific application type.

### JobAnalysis

The result of analyzing a job posting against a candidate profile.

### SkillGap

A missing or weak skill found during analysis.

### Recommendation

The suggested application decision.

Allowed values:

- Apply
- Review
- Skip

### ApplicationDraft

Generated application content for a job posting.
