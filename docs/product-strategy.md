# Product Strategy

## Purpose

Pararin Career Agent helps manage my job application process with AI support.

The first goal is not to automate everything. The first goal is to make better and faster application decisions.

## Version 1

Version 1 focuses on one workflow:

Analyze a job description and decide whether it is worth applying.

## Inputs

- Job title
- Company name
- Job description
- Job location
- Work mode
- Required skills
- Optional skills

## Outputs

- Match score
- Fit summary
- Missing skills
- Risk factors
- Application recommendation
- Suggested resume version
- Draft application message

## Out of Scope for Version 1

- Job board scraping
- Auto apply
- Browser automation
- Gmail integration
- Calendar integration
- Multi-agent architecture
- MCP integration
- Vector database
- Long-term memory

## Product Rules

- Start with manual input.
- Store every analysis result.
- Keep AI decisions explainable.
- Do not automate applications before the decision workflow is reliable.
- Add complexity only when there is a real need.
