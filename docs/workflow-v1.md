# Workflow V1

## Actor

Candidate

## Use Case

Analyze Job Posting

## Scenario

Evaluate a job posting before applying.

## System Boundary

### In Scope

- Manual job posting input
- Job analysis
- Match scoring
- Skill gap detection
- Resume version suggestion
- Draft application message generation
- Save analysis result

### Out of Scope

- Job board search
- Scraping
- Auto apply
- Browser automation
- Gmail integration
- Calendar integration

## Main Flow

1. Enter job posting.
2. Extract job details.
3. Compare job requirements with candidate profile.
4. Calculate match score.
5. Identify skill gaps.
6. Recommend apply, skip, or review.
7. Suggest resume version.
8. Generate draft application message.
9. Save analysis result.
